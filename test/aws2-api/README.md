## Description

AWS S3와 QuickSight를 연동한 NestJS API 서버입니다.

## Features

### 🗂️ S3 API
- **최신 파일 조회**: mintrend 폴더의 최신 파일에서 마지막 데이터 반환
- **날짜별 이력 조회**: 특정 날짜의 모든 mintrend 파일을 시간순 정렬하여 반환

### 📊 QuickSight API  
- **대시보드 목록 조회**: QuickSight 계정의 모든 대시보드 리스트
- **대시보드 상세 조회**: 특정 대시보드의 메타데이터 및 설정 정보
- **임베드 URL 생성**: 웹 애플리케이션에 대시보드를 임베드하기 위한 URL 생성

## Environment Variables

API 서버 실행 전에 다음 환경변수를 설정해야 합니다:

### AWS 공통 설정
```bash
AWS_REGION=ap-northeast-2                    # AWS 리전 (기본값: ap-northeast-2)
AWS_ACCESS_KEY_ID=your-access-key-id         # AWS 액세스 키
AWS_SECRET_ACCESS_KEY=your-secret-key        # AWS 시크릿 키
```

### S3 설정
```bash
S3_BUCKET_NAME=your-s3-bucket-name           # S3 버킷명 (필수)
```

### QuickSight 설정
```bash
AWS_ACCOUNT_ID=123456789012                  # AWS 계정 ID (필수)
QUICKSIGHT_NAMESPACE=default                 # QuickSight 네임스페이스 (기본값: default)
QUICKSIGHT_ALLOWED_DOMAINS=https://example.com,https://app.example.com  # 임베드 허용 도메인 (선택사항)
```

### IAM 권한 요구사항

#### S3 권한
```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "s3:GetObject",
        "s3:ListBucket",
        "s3:HeadObject"
      ],
      "Resource": [
        "arn:aws:s3:::your-bucket-name",
        "arn:aws:s3:::your-bucket-name/*"
      ]
    }
  ]
}
```

#### QuickSight 권한
```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "quicksight:ListDashboards",
        "quicksight:DescribeDashboard",
        "quicksight:GenerateEmbedUrlForRegisteredUser",
        "quicksight:GenerateEmbedUrlForAnonymousUser"
      ],
      "Resource": "*"
    }
  ]
}
```

## Project setup


## Compile and run the project

```bash

$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

```bash
$ npm run test:e2e

$ npm run test:cov
```
